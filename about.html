Como funciona

4 passos do prompt ao arquivo pronto

Do ideia ao arquivo gerado em menos de 30 segundos. Sem configuracao complicada, sem iteracao manual.

1
Idle

Crie a Tarefa

Preencha titulo, escolha a pasta, selecione o formato e descreva o que deseja criar.

2
Plan

IA Planeja

LLaMA 3.3 gera um plano tecnico estruturado. Edite no modal antes de confirmar.

3
Build

IA Constrai

Com base no plano, o modelo gera o conteudo completo e salva o arquivo na pasta escolhida.

4
Review

Revise o Resultado

Arquivo abre automaticamente no navegador para sua revisao. Aprove ou regenere.

Documentacao

API REST Completa

7 endpoints organizados em 2 modulos. Base URL: http://localhost:5000

Gestao de Tarefas

GET /api/tasks Listar tudo
POST /api/tasks Criar tarefa
PUT /api/tasks/:id Atualizar
DELETE /api/tasks/:id Remover

Pipeline de IA

POST /api/ai/plan Gerar plano
POST /api/ai/build Gerar arquivo
POST /api/ai/review Abrir arquivo
FAQ

Perguntas frequentes

Tudo que voce precisa saber para comecar a usar o KanBan AI.

E uma aplicacao web local que automatiza a criacao de arquivos (.md, .html, .js) usando Inteligencia Artificial. Voce descreve o que precisa em linguagem natural e a IA planeja, gera e salva o arquivo automaticamente, tudo organizado em um board Kanban visual.
Atualmente usa o modelo LLaMA 3.3 70B via Groq API. O modelo e excelente para geracao de codigo e documentos tecnicos, com resposta rapida e baixo custo.
Sim. O Firebase Firestore e usado como banco de dados para persistir tarefas, planos e arquivos gerados. Voce precisa criar uma conta gratuita no Firebase e configurar o Service Account no arquivo .env.
Execute 3 comandos: pip install -r requirements.txt, configure o arquivo .env com suas chaves de API, e finalmente python main.py. O app roda em http://localhost:5000/app.
Sim! O servidor roda em 0.0.0.0:5000, entao qualquer dispositivo na mesma rede Wi-Fi pode acessar via http://SEU_IP:5000/app.
Markdown (.md), HTML (.html) e JavaScript (.js). O nome do arquivo e gerado automaticamente a partir do titulo da tarefa, com sanitizacao de caracteres especiais.
